Vital Signs - 68xx Version 1.0.0 ============= Overview ======== This lab demonstrates the use of mmWave technology to accurately measure vital signs such as heart rate and breathing rate. Features ======== The following features are supported: * mmWave sensor can be placed in front or in the back of seat * Supports mmWave SDK 3.1.1.2 * Supports IWR6843 ES1.0 devices. * Ported from Driver Vital Signs xWR16xx specifically for IWR6843.
